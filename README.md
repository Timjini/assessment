# Image Web App

This application provide users with a platform to share images and comments on them.

Gems:

* Devise

* Active Storage

* Tailwind Css

* Stimulus-rails

* turbo-rails

# Getting started 
run to command to install the dependencies
`` bundle install ``

install active storage
`` $ rails active_storage:install ``

create the database

``$ rails db:create ``
migrate the database
``$ rails db:migrate ``
